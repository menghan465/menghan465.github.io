## 目的

> **为了解决研发的游戏Abyssalvage的世界观设计，即海洋废土世界观在淹没前和淹没后的人文差异，并为游戏内打捞内容提供足够的人文因素以及遗迹残留的特点，设计出可以生成世界观人文特点的工具是必要的**


---

我在做海洋废土 RPG《Abyssalvage》时用 Azgaar 生成了地图。地图分为安提梅尔纪元和珀斯梅尔纪元，这两个纪元分别表示在世界被水淹没之前和被水淹没之后。

然后我发现 D-PLACE / Ethnographic Atlas 收录的主要是 19–20 世纪记录的**小规模社会**：游牧、渔猎、刀耕火种。拿它当 prior 往地图上一套——全变成部落。而安提梅尔纪元的科技水平绝对不会这么低下。

于是我尝试读取地图的年份，根据`exports.year → 分五档：古代 / 近古 / 过渡 / 现代 / 近未来`  

- 年份≥1950：直接用 **世界银行 WDI**（GDP 人均、城市化、产业结构）+ WorldClim 气候栅格 + Natural Earth 岸线**做参考单元**  
- 年份<1950：换 premodern 功能原型（帝国/城邦/游牧邦联/封建），不再用部落志当默认  

- 年份是 2650 但 WDI 数据只到 2020 → 钳定截面 + 轻量近未来偏移（城市感 + 媒体浮一点）  ，即依然走的是**`near_future` 路径**

但这显然是错的，虽然没有新的数据集（没有 2650 年的世界银行 WDI），但可以做一个**分层未来修正**：

```
near_future (2026–210):
    WDI 2020 钳定 + near_future bias（城市感+3%、媒体感+4%）
    → 管线的当前行为，不变

far_future (2101–300):
    WDI 2020 钳定 + 更强的 future bias
    → urbanization 再 +0.1
    → 产业从 industrial/postindustrial 更多倾向到 postindustrial / "post-scarcity proxy"
    → 政体从 nation-state 倾向碎片化/城邦/公司实体
    → 海洋废土/淹没设定可以在这里 hook 一个可选修正层

deep_future (>300):
    不再用 WDI
    → 纯合成原型 + geo 约束
    → 类似 premodern 用的是功能原型，deep_future 也可以是一组「后人类/后国家」原型

```
